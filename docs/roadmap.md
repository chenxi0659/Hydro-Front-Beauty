# 前端改造路线

## 第一阶段：路径 A — 全局视觉基础

1. 建立颜色、排版、间距、圆角、阴影、状态色令牌。
2. 统一页面背景、导航、卡片、按钮、表单、表格、分页和弹窗。
3. 建立浅色和深色模式的可读性基线。

## 第二阶段：路径 B — 高价值页面渐进替换

优先顺序：

1. 控制面板与用户管理。
2. 题目管理与题目配置。
3. 比赛管理与报名/成员管理。

每一页先保留现有路由和后端数据，再由 React/Mantine 接管内容区域。

## 非目标

- 不一次性替换 Hydro 的全部 Nunjucks 模板。
- 不修改鉴权、判题、数据库或业务接口。
- 不立即基于 UI Next 重写全部页面；待其页面覆盖度和插件接口稳定后再评估。
